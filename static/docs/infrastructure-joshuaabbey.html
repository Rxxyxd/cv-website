DNS Configuration

I used he.net to point the name server from namecheap.com to he.net as a preferred way to change the DNS settings of the domain. I wanted to make sure that everything I wanted to do worked smoothly so I used dns.he.net. Figure 1 shows the DNS settings I used for this project:

Obviously all of the entries of type NS directed the domain to he.net but CNAME of name “www.joshuaabbey.com” directed it to the hostname joshuaabbey.com and the entry of A type “joshuaabbey.com” directed to the hostname 178.79.191.194. The reason why I did this was so that if joshuaabbey.com had the subdomain of www it would redirect to the hostname joshuaabbey.com and ofc joshuaabbey.com would have to redirect to the server IP.
